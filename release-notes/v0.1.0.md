# OpsProbe v0.1.0

## Summary

`v0.1.0` establishes the foundation for the open source edition of OpsProbe.

This release defines the product boundary, introduces the first Tauri desktop application skeleton, and sets up the repository structure required for the local-first inspection MVP.

## Included Milestone

- `0.1.0` Foundation

## Key Changes

- Added repository-level product documentation and MVP scope
- Added versioning and release planning documents
- Bootstrapped the Tauri desktop app in `apps/desktop`
- Added root workspace configuration
- Added placeholder package structure for reusable modules
- Replaced the default Tauri welcome screen with an OpsProbe-oriented foundation view

## Known Limits

- No real inspection execution yet
- No SSH connectivity flow yet
- No report generation yet
- Shared packages are placeholders for upcoming versions

## Next Target

- `0.2.0` First Inspection Flow
