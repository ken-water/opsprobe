# OpsProbe v0.8.3

Released: 2026-06-05

`v0.8.3` closes the current `0.8.x` exploration line with stronger functional confidence and more repeatable release validation.

## Highlights

- Adds a local-service CLI functional test for asset save, inspection preview, and HTML report export
- Adds a dedicated smoke script for the same asset-preview-report workflow before broader release checks
- Updates release validation guidance so `0.8.3` reflects the first automated functional slice in OpsProbe

## Why this release exists

The previous `0.8.x` checkpoints improved inspection depth and report usefulness, but the release line still depended too much on manual confidence. `0.8.3` strengthens the path from saved asset to exported report so regressions in a core local workflow are easier to catch before a tag is published.
