# OpsProbe v0.12.4

`v0.12.4` closes the next pre-stable packaging checkpoint by proving that the current blocker is not only a Windows installer gap. The same stale vendored Cargo source now blocks the current Linux packaged build as well, and both failures reduce to the same missing `linux-raw-sys 0.12.2` vendored input on this machine.
