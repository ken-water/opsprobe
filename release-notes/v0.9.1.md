# OpsProbe v0.9.1

Released: 2026-06-07

`v0.9.1` continues the `0.9.x` service-depth line with a more credible Redis inspection workflow for recurring SMB operations.

## Highlights

- Adds Redis depth checks for memory pressure, persistence risk, blocking risk, and eviction or rejection risk
- Implements real SSH execution paths for Redis process, listener, runtime, replication, and the new operational checks
- Extends CLI functional validation and smoke validation so the Redis preview template path is exercised directly

## Why this release exists

`0.9.0` proved that OpsProbe could deepen one service-role workflow meaningfully with MySQL. `0.9.1` applies the same standard to Redis, where memory ceilings, persistence posture, blocked clients, and eviction behavior often matter more than simple process reachability in day-to-day SMB operations.
