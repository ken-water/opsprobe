# OpsProbe v0.7.3

Released: 2026-06-05

`v0.7.3` adds a structured early-user feedback workflow so outside users can submit actionable inspection, report, and workflow requests instead of generic comments.

## Highlights

- Adds GitHub issue templates for inspection needs, report feedback, and workflow friction
- Adds a feedback guide with direct links and concrete prompts
- Adds desktop feedback entry points so users can jump from product usage to structured roadmap input
- Updates repository entry points to make feedback routing obvious

## Why this release exists

OpsProbe now needs higher-signal outside feedback, not just more raw reactions. This release makes it easier to capture real user needs in a format that can guide roadmap decisions.

## Validation

- `npm run desktop:typecheck`
- `npm --workspace @opsprobe/desktop run build`
