# OpsProbe v0.9.2

Released: 2026-06-07

`v0.9.2` continues the `0.9.x` service-depth line with a more credible Nginx inspection workflow for recurring SMB operations.

## Highlights

- Adds Nginx depth checks for upstream hints, error-log risk, TLS posture, and config-drift hints
- Implements real SSH execution paths for Nginx process, config validation, virtual-host inventory, and the new operational checks
- Extends CLI functional validation and smoke validation so the Nginx edge preview template path is exercised directly

## Why this release exists

`0.9.0` and `0.9.1` deepened database and cache workflows. `0.9.2` applies the same standard to Nginx, where upstream drift, repeated error-log signals, TLS posture, and untracked config changes often matter more to operators than simple process reachability.
