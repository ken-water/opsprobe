# OpsProbe v0.8.1

Released: 2026-06-05

`v0.8.1` continues the `0.8.x` exploration line with deeper Kubernetes node inspection coverage and more actionable report wording for node-side findings.

## Highlights

- Adds deeper Kubernetes node checks for node summary, static pod inventory, kubelet health, and node pressure signals
- Improves remediation wording for Kubernetes findings so the next operator action is clearer
- Adds `Action focus` lines to report output for faster scanning of abnormal and priority items

## Why this release exists

`0.8.0` improved service-role depth for Docker and other common Linux workloads. `0.8.1` extends that same principle to Kubernetes nodes: stay local-first, stay SSH-first, and make recurring operational review more useful without requiring a hosted control plane.
