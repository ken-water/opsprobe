# OpsProbe v0.9.4

Released: 2026-06-07

`v0.9.4` continues the `0.9.x` service-depth line with install, upgrade, migration, and regression hardening for recurring SMB operations.

## Highlights

- Adds migration-focused regression coverage for config export and import, credential masking, template fallback, and stopped-state recovery
- Introduces a release-readiness gate that blocks publishing until version files, release notes, desktop copy, and smoke validation all align
- Extends release smoke validation to cover manager report export and config export/import rebind behavior

## Why this release exists

`0.9.0` through `0.9.3` improved service depth and report quality. `0.9.4` focuses on release credibility: safer migration behavior, clearer local-state recovery, and stronger gates so install and upgrade confidence can be evaluated honestly before any `1.0.0` decision.
