# OpsProbe v0.8.0

Released: 2026-06-05

`v0.8.0` starts the `0.8.x` exploration line with deeper Docker host inspection coverage and the first release-ready packaging of the post-validation workflow improvements.

## Highlights

- Adds deeper Docker host checks for runtime summary, image inventory, and abnormal container evidence
- Includes the earlier `0.8.0` exploration work for nginx, MySQL, Redis, and the automated test foundation
- Defines the `0.8.1` to `0.8.3` follow-up checkpoints so the rest of the `0.8.x` line is explicit and GitHub-aligned

## Why this release exists

The `0.7.x` line proved the product is easier to understand and set up, but it did not yet prove that the built-in workflows are deep enough for recurring real operations work. `0.8.0` is the first checkpoint in that next stage: less emphasis on new surfaces, more emphasis on higher-quality inspection evidence for common Linux service roles.
