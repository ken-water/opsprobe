# OpsProbe v0.11.10

## Summary

`v0.11.10` unifies the sticky top status area and global feedback banners so the desktop speaks with a more consistent state and action language across workspaces.

## Included Milestones

- `0.11.x` desktop operator-flow simplification

## Key Changes

- replace the older topbar badge mix with a clearer status summary plus runtime and mode chips
- change transient feedback headings to clearer intent labels such as `Working`, `Done`, and `Action Needed`
- align topbar and feedback styling so long-running actions, successes, and errors feel more consistent

## Scope Boundary

Included in `0.11.10`:

- topbar state hierarchy cleanup
- global feedback wording and visual consistency improvements
- matching UI regression validation

Deferred to the next version:

- further desktop simplification outside the shared feedback system
- packaged acceptance and stable-decision evidence work

## Known Limits

- `0.11.10` improves state readability, but the overall desktop still needs more refinement before a `1.0.0` stability claim

## Next Target Version

- `0.11.11`
