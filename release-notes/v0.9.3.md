# OpsProbe v0.9.3

Released: 2026-06-07

`v0.9.3` continues the `0.9.x` service-depth line with correlated host-and-service reporting for recurring SMB operations.

## Highlights

- Adds correlated `priorityActions` that group related host and service abnormalities into clearer remediation work items
- Introduces queue-style action metadata including rank, urgency, rationale, and related-signal summaries in HTML and manager PDF reports
- Extends automated correlation coverage across Nginx, MySQL, Redis, and Kubernetes report scenarios

## Why this release exists

`0.9.0` through `0.9.2` improved service-specific inspection depth. `0.9.3` turns those isolated findings into a more usable action queue, so operators and managers can see what to fix first and why the grouped signals belong to the same repair track.
