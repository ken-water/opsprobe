# OpsProbe v0.9.0

Released: 2026-06-07

`v0.9.0` starts the `0.9.x` service-depth line with a more credible MySQL inspection workflow for recurring SMB operations.

## Highlights

- Adds MySQL depth checks for connection pressure, replication hints, slow-query posture, and temporary disk-table spill risk
- Implements real SSH execution paths for MySQL process, listener, runtime configuration, schema inventory, and the new operational checks
- Extends CLI functional validation and smoke validation so the MySQL preview template path is exercised directly

## Why this release exists

The `0.8.x` line improved release discipline and local workflow confidence, but OpsProbe still needed deeper service-role inspection value before a stable milestone would be credible. `0.9.0` focuses on MySQL because it is a high-frequency SMB workload and a strong test case for whether OpsProbe can move beyond basic service reachability into meaningful operator guidance.
