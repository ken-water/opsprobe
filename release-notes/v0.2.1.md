# OpsProbe v0.2.1

## Summary

`v0.2.1` adds password-based SSH support on top of the first inspection flow introduced in `v0.2.0`.

## Included Scope

- Password-mode SSH connection testing
- Password-mode SSH execution for Linux baseline checks

## Key Changes

- Added `sshpass`-backed password authentication for the Tauri SSH command path
- Kept `private-key` mode intact as the original path
- Updated the desktop credential input to use a password field when password mode is selected
- Added explicit validation when `sshpass` is not available locally

## Known Limits

- Password mode depends on `sshpass` being installed on the local machine
- Secrets are still entered directly in the desktop UI and are not yet backed by a dedicated secret store
