# OpsProbe v0.7.2

Released: 2026-06-05

`v0.7.2` adds report variants so early users can compare what operators need for execution versus what managers need for quick decision-making.

## Highlights

- Adds operator and manager report audiences shared across HTML and PDF export
- Persists the selected audience in desktop settings so repeated exports stay consistent
- Introduces a concise manager summary layout with key highlights and priority actions
- Keeps the detailed operator layout with evidence and remediation depth for hands-on troubleshooting

## Why this release exists

Different readers want different report density. This release makes that tradeoff explicit so external users can say which structure is actually useful instead of reacting to a single report format.
